Учебные проекты по макетам от
«HTML Academy»

В учебных проектах мной использованы:

- стандарты HTML
- сервис валидации HTML-файлов
- файл normalize.css - для исправления браузерных умолчаний
- автопрефиксер CSS - для добавления префиксов перед экспериментальным или нестандартным CSS-свойством
- компрессоры HTML, CSS и JS и растровых изображений

- семантическая верстка
- атрибуты "aria-label" для улучшения доступности

- технология "flex" для создания сеток/микросеток
- верстка страниц под различную ширину окна в 2-х проектах (Sedona [768/1200], Gllacy [900/1200])

- GoogleFonts и локальные шрифты
- сервис для создания собственных шрифтов из svg-иконок

- якорные ссылки
- паттерны для полей ввода

- JS для открытия/сокрытия модальных окон
- Google API для подключения карт к страницам (Gllacy)

- CSS-анимация для "оживления" слайдера
- CSS-transition - для плавной смены фона страницы
- средства CSS для переключения слайдов
